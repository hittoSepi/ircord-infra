Quick Overview

IRCord is an open-source, end-to-end encrypted messaging platform that combines the simplicity of IRC with modern security. The server never sees your message content — all encryption happens on your device.

Key Features

Feature Description
E2E Encryption Signal Protocol for DMs, Sender Keys for channels
Voice Rooms Push-to-talk and voice activation with WebRTC
Channels & DMs IRC-style channels with modern encrypted direct messages
Plugin System JavaScript plugins: bots, client extensions, and server extensions
Cross-Platform Windows, Linux (TUI), and Android (native)
Self-Hostable Run your own server on a Raspberry Pi or VPS
