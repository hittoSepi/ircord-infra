Host Your Own Server

Want to run your own IRCord server? It's easy!

  1. Download the server binary for your platform
  2. Generate TLS certificates
  3. Edit the configuration file
  4. Run the server
# Quick start
./ircord-server --config server.toml

# Enable public listing (optional)
# Edit server.toml and set:
# [server]
# public = true
#
# [directory]
# enabled = true
# server_name = "My Awesome Server"
# description = "A friendly place to chat"
